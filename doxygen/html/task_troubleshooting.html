Deletion Error

Problem:This error message shows up when you try to delete data that is used in another table.

Cause: The error occurs because the data you are trying to delete is referenced in another table, creating a dependency between the two tables.

Fix:

  1. Find the record where that data is used. (Sometimes it might be multiple records in different tables)
  2. Update these records with new information
  3. Try deleting again

Note:
This error message only shows up when you try to delete data but it is used elsewhere.
