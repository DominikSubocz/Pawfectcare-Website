Deleting Pets

To delete pet from the system, follow these steps:

  1. Navigate to Pet List
  2. Select Pet to Delete
  3. Click Delete Button
  4. Return to List
  5. After submitting, you will be redirected back to the respective list.
Note:
Error message will show up if the data is used in another table and cannot be removed.
To resolve this: simply find where the data is used, update that record where the data is used to new one, then try deleting again.
