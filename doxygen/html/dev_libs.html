1. Bootstrap

Bootstrap

Description: Bootstrap is a popular front-end framework for building responsive and mobile-first websites. It provides a collection of CSS and JavaScript components that simplify the process of creating user interfaces.

Usage in Project: Bootstrap was used extensively in this project to ensure a consistent and responsive design across different devices and screen sizes. It enabled the creation of layouts, forms, buttons, navigation elements, and other UI components with minimal custom styling. It was also used to remaster/improve some of the clunky visual elements that were implemented with CSS before using bootstrap.

2. Doxygen

Doxygen

Description: Doxygen is a documentation generator tool that is commonly used for C++, C, Java, Python, and other programming languages. It extracts documentation from source code comments and generates various output formats, including HTML, PDF, and LaTeX.

Usage in Project: Doxygen was utilized to automatically generate documentation for the project's source code. By adding specially formatted comments (e.g., using Doxygen-style tags), the developers could produce structured and organized documentation directly from the codebase. This documentation is crucial for understanding the functionality and usage of various components within the project.
