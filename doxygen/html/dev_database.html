3. Database SQL

Database SQL

Database schema for this project consists of several key tables that store essential data for our application. Below is a summary of the main tables and their purposes:

  • Addresses: Manages address information, including address IDs, lines, city, county, and postcode.
  • Users: Stores user information, including user IDs, usernames, email addresses, and user roles.
  • Doctors: Manages doctor details such as doctor IDs, first names, last names, and contact numbers.
  • Juniors: Tracks information about junior players, including junior IDs, addresses, names, and health issues.
  • Guardians: Stores guardian details such as guardian IDs, addresses, names, contact numbers, and relationships.
  • Junior Associations: Manages associations between juniors, guardians, and doctors.
  • Players: Manages player details such as player IDs, addresses, names, contact information, and next of kin details.
  • Members: Stores member information, including member IDs, addresses, names, contact information, and email addresses.
  • Skills: Manages various skills categorized by type and name.
  • Squads: Tracks squad details including squad IDs, names, and levels.
  • Junior Skills: Manages skill levels and comments for juniors within squads.
  • Player Skills: Tracks skill levels and comments for players within squads.
  • Positions: Stores rugby positions.
  • Junior Positions: Manages positions for junior players.
  • Player Positions: Manages positions for players.
  • Games: Tracks game information including game IDs, squad IDs, names, opposition teams, locations, and results.
  • Coaches: Stores coach details such as coach IDs, user IDs, names, contact information, and email addresses.
  • Sessions: Manages session details including session IDs, coach IDs, squad IDs, names, locations, and timings.
  • Game Halves: Tracks game halves including game half IDs, game IDs, team names, scores, and comments.
  • Training Details: Stores training details such as session IDs, coach IDs, squad IDs, skills, activities, and player attendance.
