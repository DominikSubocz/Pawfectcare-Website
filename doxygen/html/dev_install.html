Installation Instructions

  1. Download the latest version of the website from the GitHub Releases page.
  2. Extract the 'PawfectCare' folder from the downloaded .zip file into the 'xampp/htdocs' folder on your local machine.
  3. Open XAMPP control panel and start both Apache and MySQL services.
  4. Open and copy the content of 'database.sql' file and paste it in the SQL tab (it should work firs time, but please let me know if there are errors)
  5. Open a web browser and navigate to the localhost URL (or the URL you configured for XAMPP)
  6. From the list of websites, locate and click on 'PawfectCare' to access the website.
  7. That's it! You can now use the website.
